Accessibility Statement

We are committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards to satisfy WCAG 2.1 Level AA compliance.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA. This portfolio is fully conformant with WCAG 2.1 Level AA.

Accessibility Features Available

  • Keyboard Accessibility: Full navigation support using standard keyboard triggers.
  • Text Resizing: Zoom and scale the layout proportionally without breaks.
  • Text Spacing Adjustment: Toggles expanded spacing for better readability.
  • High Contrast & Themes: High contrast, grayscale, and light/dark themes.
  • Visual Guides: Focus outlines, dyslexia-friendly font (Lexend), customized large cursor, reading guides, and reading masks.
  • Reduced Motion: Ability to freeze GSAP, Three.js animations, and CSS transitions.
